# Проект САККОВ - блок схема проекта

## Блок-схема процесса

```mermaid
graph TD;
    A[Подготовка данных] --> B[Обращение в БД для получения данных звонков]
    B --> C[Получение расшифровки звонка]
    C --> D[Проверка данных в БД]
    D --> E[Запись информации о звонке в таблицу call_scores]
    E --> F[Кэширование и сбор данных]
    F --> G[Кэширование полученных данных]
    G --> H[Проверка полноты данных]
    H --> I[Анализ звонков с использованием ИИ]
    I --> J[Выбор чек-листа]
    J --> K[Проведение анализа по чек-листу]
    K --> L[Категоризация звонка]
    L --> M[Критерии оценки по каждой категории]
    M --> N[Запись результатов анализа]
    N --> O[Запись категории и заключения]
    O --> P[Сравнение динамики по звонку за предыдущие периоды]
    P --> Q[Обратная связь и обучение операторов]
    Q --> R[Доставка заключений и рекомендаций]
    R --> S[Мониторинг и логирование]
    S --> T[Автоматизация развертывания и CI/CD]
    T --> U[Управление пользователями и доступом]
    U --> V[Отчеты и аналитика]
    V --> W[Обратная связь и улучшения на основе данных]
Тест
